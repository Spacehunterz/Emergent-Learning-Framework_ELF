# Recording Notes

- Keep phrases 2-4 seconds max
- Natural, conversational tone
- Varied inflection (not monotone)
- Process with: `ffmpeg -i input.mp4 -vf "scale=766:-1,crop=766:1135,pad=928:1376:(928-766)/2:(1376-1135)/2:black" -r 24 output.mp4`
- Extract audio: `ffmpeg -i video.mp4 -vn -acodec libmp3lame -b:a 192k audio.mp3`
