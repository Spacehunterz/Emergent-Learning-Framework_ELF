# Invisible Affordances Fail

**Heuristic:** Users cannot interact with UI elements they cannot see. Edge-based resize zones without visual indicators are undiscoverable.

**Why:** Always add visible affordances (icons, handles, triangles) for interactive elements.

**Source:** claudex resize handle - user couldn't find edge resize until triangle added
**Confidence:** 0.8
