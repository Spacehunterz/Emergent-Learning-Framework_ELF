# Failure: Acting on Discussion as if it were Command

**Date:** 2025-12-02
**Domain:** communication, process
**Severity:** Medium

## What Happened

User said "briefly I think we can consolidate many commands into the swarm command"
- This is DISCUSSION language ("I think", "briefly")
- I immediately started renaming the plugin
- User had to interrupt and correct me

## Pattern Recognition

**Discussion phrases (DON'T ACT):**
- "I think we could..."
- "Let's discuss..."
- "briefly..."
- "what if..."
- "imagine..."

**Action phrases (DO ACT):**
- "Do it"
- "Go ahead"
- "Make it"
- "Rename it to X"
- "Change X to Y"
- Imperative verbs with specific targets

## Root Cause

- Over-eagerness to be helpful
- Treating ideation as instruction
- Not distinguishing discussion from command

## Heuristic

> Discussion â‰  Command. When user uses exploratory language ("I think", "what if", "imagine"), STAY IN CONVERSATION. Only act on explicit imperatives ("do it", "make it", "go ahead").
